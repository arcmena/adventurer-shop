## Getting started

Clone this repo:

```bash
git clone https://github.com/arcmena/adventurer-shop.git
```

Install the dependencies:

```bash
npm install
```

or simply:

```bash
yarn
```

Run to start: 

```bash
yarn start
```

> **Note:** The server will start at http://localhost:3000 by default.
